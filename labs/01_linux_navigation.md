# 01 — Linux Navigation

## Goal

Develop familiarity with the Linux command line and file system structure.

Understanding basic navigation commands is essential for cybersecurity professionals because many security tools, investigations, and servers operate within Linux environments.

---

## What I Did

* Opened a Linux terminal environment.
* Identified my current working directory.
* Listed files and folders within directories.
* Navigated between directories.
* Created files and folders.
* Viewed file contents.
* Verified command output.

---

## Commands Used

### Display Current Directory

```bash
pwd
```

Displays the current working directory.

### List Directory Contents

```bash
ls
```

Displays files and folders in the current directory.

### Change Directories

```bash
cd
```

Navigates between directories.

### Create a Directory

```bash
mkdir test_directory
```

Creates a new directory.

### Create a File

```bash
touch test_file.txt
```

Creates a new file.

### View File Contents

```bash
cat test_file.txt
```

Displays file contents.

---

## What I Observed

* Linux organizes information through a hierarchical file system.
* Navigation occurs through directories and subdirectories.
* Commands execute quickly and provide immediate feedback.
* Many administrative and security tasks rely on command-line access.
